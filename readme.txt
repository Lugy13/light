api.message.loc - папка содержащая php часть.
Используется как API сервис. Её необходимо развернуть
на локальном сервере (порт 3000), чтобы не править адрес в коде.

light.loc - папка фронта (Angular2).
Эту часть можно запустить с помощью light server.

light_it.sql - дамп базы данных с тестовыми значениями.
